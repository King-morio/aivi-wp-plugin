# AiVI - AI Visibility Inspector

AiVI is a content intelligence and correction layer for AI-driven search. It helps WordPress publishers review extraction, structure, schema, trust, and answer-readiness issues before publishing.

## Download

- Latest release: `v1.0.30`
- Release ZIP: `ai-visibility-inspector-1.0.30.zip`
- GitHub Releases: `https://github.com/King-morio/aivi-wp-plugin/releases`

## What AiVI Does

- analyzes WordPress content for AI visibility before publishing
- surfaces extraction, trust, structure, schema, and answer-readiness gaps
- supports both Gutenberg and Classic Editor workflows
- provides guided sidebar review and in-context overlay support

## Install

1. Download the latest release ZIP.
2. In WordPress admin, go to `Plugins > Add New > Upload Plugin`.
3. Upload the ZIP and activate **AiVI - AI Visibility Inspector**.

## Editing Note

AiVI does not auto-publish or auto-apply content into your editor. Review the guidance it gives you, then copy and paste any approved revisions into the matching WordPress block before clicking `Update` or `Publish`.
