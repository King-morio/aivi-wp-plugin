# AiVI - AI Visibility Inspector

AiVI is a content intelligence and correction layer for AI-driven search. It analyzes WordPress content for extraction, trust, structure, schema, and answer readiness, then guides fixes before publication.

## Download

- Latest release: `v1.0.24`
- Release ZIP: `ai-visibility-inspector-1.0.24.zip`
- GitHub Releases: `https://github.com/King-morio/aivi-wp-plugin-public/releases`

## What AiVI Does

- analyzes WordPress content for AI visibility before publishing
- highlights extraction, trust, structure, schema, and answer-readiness gaps
- supports both Gutenberg and Classic Editor workflows
- provides guided sidebar review and in-context overlay editing support

## Install

1. Download the latest release ZIP.
2. In WordPress admin, go to `Plugins > Add New > Upload Plugin`.
3. Upload the ZIP and activate **AiVI - AI Visibility Inspector**.

## Editing Note

AiVI does not auto-publish your edits. The current overlay flow lets you review revisions in AiVI, then copy and paste them into the matching WordPress block before clicking `Update` or `Publish`.
