# GitHub Release 1.0.24

## Title

`AiVI 1.0.24`

## Suggested Tag

`v1.0.24`

## Summary

AiVI `1.0.24` is the clean submission-ready build prepared for WordPress.org review and pre-launch distribution.

## Release Notes

### Highlights

- polished the WordPress.org-facing plugin description, author, and contributor metadata
- kept the safer manual-copy overlay workflow intact
- preserved the wider editorial stage while restoring a slimmer, less intrusive block actions menu
- kept sidebar polish, extractability hardening, and post-body image fidelity improvements in the current release line

### Install

1. Download `ai-visibility-inspector-1.0.24.zip`
2. In WordPress admin, go to `Plugins > Add New > Upload Plugin`
3. Upload the ZIP and activate AiVI

### Notes

- AiVI supports both Gutenberg and Classic Editor workflows
- the current overlay flow is manual copy/paste into the matching WordPress block before you click `Update` or `Publish`
- WordPress.org listing banners, icons, and screenshots are staged separately for post-approval SVN upload
